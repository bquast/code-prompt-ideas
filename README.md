### what is CodeBench?

CodeBench is a **side-by-side code battle arena** focused on vanilla JavaScript.

Users are shown:
- One coding prompt
- Two different solutions generated by top models (currently **GPT-4o** vs **Grok-4.20**)

They vote for the better one **after seeing live execution results**.

### goal

To create a high-quality, **public open-source dataset** of human preferences on code quality, style, correctness, and creativity.

Every vote is recorded and becomes training data for future code models.

### how it works

1. A random prompt is selected
2. GPT-4o and Grok-4.20 generate solutions independently
3. Both codes are executed live in the browser with test cases
4. You compare and vote for the better version
5. Votes + generations are stored in D1

### tech Stack

- **Cloudflare Pages** (static + Functions)
- **D1** (SQLite) — stores prompts, generations, battles and votes
- pure **vanilla HTML/CSS/JS** (no build step)
- OpenAI-compatible API calls (GPT-4o + Grok)

### dataset

All data is open and will be made publicly available (prompts, model outputs, and human votes).

This is one of the first community-driven preference datasets focused specifically on **code generation quality**.
